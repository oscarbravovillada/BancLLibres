1. Accés al sistema

  1. Obre el navegador i accedeix a la URL del sistema (facilitada pel coordinador TIC).
  2. Introdueix el teu nom d'usuari i contrasenya.
  3. Prem Inicia sessió.
Nota: Sols veuràs els alumnes assignats a les teves classes. Si no veus un alumne esperat, contacta amb l'administrador.

2. Panell principal (inici)

En accedir al sistema veuràs el panell d'inici, dissenyat per mostrar-te d'un cop d'ull tota la informació de les teues classes.

2.1 Bàner de benvinguda

A la part superior apareix una salutació personalitzada amb el teu nom, la data del dia i el nombre de classes que tens assignades.

2.2 Estadístiques ràpides

Una fila de píndoles mostra els totals globals del centre: exemplars en préstec, perduts, incidències, etc. Les píndoles amb valors problemàtics (perduts > 0, imports per pagar) s'il·luminen en color.

2.3 Les teues classes

Cada classe assignada apareix com una targeta compacta amb:

  • Nombre d'alumnes actius de la classe.
  • Préstecs actius en aquest moment.
  • Incidències (en taronja/roig si n'hi ha).
  • Botó Veure alumnes → accés directe a la llista d'alumnes d'eixa classe.
Important: Sols veuràs les classes assignades a tu. Si esperaves veure una classe i no apareix, contacta amb l'administrador.

2.4 Accions ràpides

Al costat dret trobaràs els botons de les accions més habituals:

BotóAccés directe a…
Assignar lot de llibresPàgina de classes per iniciar l'assignació
Registrar devolucióFormulari de devolució per codi d'exemplar
IncidènciesLlistat i creació d'incidències
Llista de préstecsTots els préstecs actius de les teues classes

2.5 Activitat recent

A la part inferior, una taula mostra els últims 6 préstecs de les teues classes (data, alumne, classe, codi d'exemplar i estat).

3. Gestió d'alumnes

3.1 Veure la llista d'alumnes

Ves a Alumnes → Llista d'alumnes. Veuràs una taula amb:

  • Nom i cognoms, classe, número de préstecs actius.
  • Badges de color: Sense préstecs, Amb préstecs, Amb incidències.

3.2 Fitxa de l'alumne

Fes clic sobre el nom d'un alumne per obrir la seva fitxa. Des d'aquí pots:

  • Veure els exemplars prestats actualment.
  • Accedir a les opcions: Assignar lot, Afegir optativa, Registrar devolució, Registrar incidència, Reenviar document, Historial.

4. Assignar llibres (préstecs)

4.1 Assignar el lot de matèries comunes

  1. Obre la fitxa de l'alumne.
  2. Prem Assignar lot.
  3. Selecciona el lot corresponent al curs de l'alumne.
  4. Prem Confirmar assignació.
  5. Es generarà automàticament un albarà de préstec en PDF i s'enviarà per correu a la família (si hi ha adreça configurada).

4.2 Afegir un llibre d'optativa

  1. Obre la fitxa de l'alumne.
  2. Prem Afegir llibre d'optativa.
  3. Selecciona la matèria i el curs de l'optativa.
  4. El sistema assignarà l'exemplar disponible i generarà l'albarà.
Consell: Comprova prèviament que hi ha exemplars disponibles a Llibres → Exemplars filtrant per la matèria desitjada.

5. Registrar devolució

  1. Obre la fitxa de l'alumne.
  2. Prem Registrar devolució.
  3. Per a cada exemplar, selecciona l'estat:
    • Pendent de retorn — l'alumne no ha portat el llibre avui, queda marcat com a pendent.
    • Retornat — el llibre s'ha lliurat en bon estat.
    • Perdut — el llibre no es pot recuperar; es genera una incidència automàticament.
  4. Prem Confirmar devolució.
  5. Es generarà l'albarà de devolució i s'enviarà per correu.
Préstecs pendents: Un exemplar en estat pendent continua associat a l'alumne. Quan el retorni, torna a fer el procés de devolució i selecciona Retornat.

6. Incidències

6.1 Registrar una incidència manualment

  1. Obre la fitxa de l'alumne i prem Registrar incidència.
  2. Selecciona l'exemplar afectat.
  3. Tria el tipus: Deteriorament lleu, Deteriorament greu, Pèrdua o Altres.
  4. Omple la descripció i, si escau, l'import a pagar.
  5. Prem Guardar incidència.

6.2 Marcar una incidència com a pagada

  1. Ves a Incidències al menú.
  2. Localitza la incidència (filtra per alumne si cal).
  3. Prem Marcar com a pagat i confirma.

7. Historial

Per veure totes les accions registrades d'un alumne (assignacions, devolucions, incidències i correus enviats), obre la fitxa de l'alumne i prem el botó Historial. Veuràs la línia de temps completa d'eixe alumne ordenada per data.

8. Reenviar documents per correu

  1. Obre la fitxa de l'alumne.
  2. Prem Reenviar document.
  3. Selecciona el document (albarà de préstec, devolució o incidència) i confirma.
  4. El sistema reenviarà el PDF a l'adreça de correu de la família.
Si la família no té adreça de correu configurada, el botó de reenviar no funcionarà. Contacta amb l'administrador per actualitzar les dades.

9. Préstecs pendents de retorn

Un exemplar marcat com a pendent apareix a la fitxa de l'alumne i a la llista de préstecs amb l'etiqueta Pendent. Per tancar-lo:

  1. Quan l'alumne porti el llibre, obre la fitxa i prem Registrar devolució.
  2. Selecciona Retornat per a l'exemplar en qüestió i confirma.

10. Preguntes freqüents

PreguntaResposta
No veig un alumne a la llistaL'alumne pot no estar assignat a la teva classe. Avisa l'administrador.
No es genera el PDFComprova que hi ha espai en el servidor. Avisa l'administrador si el problema persisteix.
El correu no arriba a la famíliaComprova l'adreça a la fitxa de l'alumne. Si és correcta, el servidor SMTP pot estar fallant — avisa l'administrador.
M'apareix "Accés denegat"Estàs intentant accedir a un alumne d'una altra classe. Sols pots gestionar els alumnes assignats a tu.
He comès un error en la devolucióContacta amb l'administrador per corregir-ho manualment a la BD.
Puc editar el catàleg de llibres?Sí, professors i admin poden afegir/editar llibres i exemplars.
