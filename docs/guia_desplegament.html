2. Instal·lació de XAMPP

# Descarregar instal·lador (versió 8.2.x)
wget https://sourceforge.net/projects/xampp/files/XAMPP%20Linux/8.2.12/xampp-linux-x64-8.2.12-0-installer.run

# Donar permisos i executar
chmod +x xampp-linux-x64-8.2.12-0-installer.run
sudo ./xampp-linux-x64-8.2.12-0-installer.run

# Iniciar serveis
sudo /opt/lampp/lampp start
Nota: XAMPP s'instal·la a /opt/lampp/. Els projectes web han d'anar a /opt/lampp/htdocs/.

2.1 Configurar XAMPP per iniciar automàticament

sudo nano /etc/systemd/system/xampp.service

Contingut del fitxer:

[Unit]
Description=XAMPP
After=network.target

[Service]
Type=forking
ExecStart=/opt/lampp/lampp start
ExecStop=/opt/lampp/lampp stop
RemainAfterExit=yes

[Install]
WantedBy=multi-user.target

sudo systemctl daemon-reload
sudo systemctl enable xampp
sudo systemctl start xampp

3. Desplegar el codi

3.1 Via Git (recomanat)

cd /opt/lampp/htdocs
git clone https://github.com/oscarbravovillada/BancLLibres.git

3.2 Via còpia manual

# Crear carpeta i copiar els fitxers
sudo mkdir -p /opt/lampp/htdocs/BancLLibres
sudo cp -r /ruta/al/projecte/* /opt/lampp/htdocs/BancLLibres/

# Assignar permisos
sudo chown -R daemon:daemon /opt/lampp/htdocs/BancLLibres
sudo chmod -R 755 /opt/lampp/htdocs/BancLLibres

3.3 Crear carpetes de pujada

sudo mkdir -p /opt/lampp/htdocs/BancLLibres/uploads/albarans
sudo chmod 777 /opt/lampp/htdocs/BancLLibres/uploads/albarans

4. Configurar la base de dades

4.1 Crear la BD i l'usuari

# Accedir a MySQL (password root de XAMPP per defecte és buit)
sudo /opt/lampp/bin/mysql -u root

-- Dins MySQL:
CREATE DATABASE banclLibres CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'banclLibres'@'localhost' IDENTIFIED BY 'la_teva_contrasenya';
GRANT ALL PRIVILEGES ON banclLibres.* TO 'banclLibres'@'localhost';
FLUSH PRIVILEGES;
EXIT;

4.2 Importar l'esquema

sudo /opt/lampp/bin/mysql -u banclLibres -p banclLibres < /opt/lampp/htdocs/BancLLibres/database/schema.sql
L'esquema ja inclou l'usuari admin per defecte (password: Admin1234!). Canvia'l immediatament després del primer inici de sessió.

5. Configurar config.php

Copia la plantilla i edita-la:

cp /opt/lampp/htdocs/BancLLibres/config/config.example.php \
   /opt/lampp/htdocs/BancLLibres/config/config.php
nano /opt/lampp/htdocs/BancLLibres/config/config.php

Paràmetres a ajustar:

Mai facis git commit del fitxer config/config.php — conté credencials. Ja està al .gitignore.

6. Verificar que funciona

  1. Obre el navegador i accedeix a http://localhost/BancLLibres/public/
  2. Hauries de veure la pàgina de login.
  3. Inicia sessió amb admin / Admin1234!
  4. Comprova que el menú mostra: Alumnes, Préstecs, Llibres, Incidències, Historial, Usuaris.
  5. Canvia la contrasenya de l'admin des del menú Usuaris.

7. Actualitzar el codi (desplegaments futurs)

cd /opt/lampp/htdocs/BancLLibres
git pull origin main

# Si hi ha canvis d'esquema (schema_vX.sql):
sudo /opt/lampp/bin/mysql -u banclLibres -p banclLibres < database/schema_vX.sql

8. Còpies de seguretat

8.1 Script de backup diari (crontab)

# Editar crontab de root
sudo crontab -e

# Afegir línia per backup a les 02:00h cada dia:
0 2 * * * /opt/lampp/bin/mysqldump -u banclLibres -p'la_teva_contrasenya' banclLibres \
  | gzip > /var/backups/banclLibres_$(date +\%Y\%m\%d).sql.gz

8.2 Retenir 30 dies

# Afegir al crontab (neteja els backups de més de 30 dies):
0 3 * * * find /var/backups/ -name 'banclLibres_*.sql.gz' -mtime +30 -delete
