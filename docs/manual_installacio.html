1. Requisits previs

  • Ordinador o servidor amb Ubuntu 22.04 LTS (o versió posterior).
  • Accés a terminal amb privilegis sudo.
  • Connexió a Internet per descarregar XAMPP i el codi.
  • Compte de correu SMTP (p. ex. Office 365) per a les notificacions a famílies.

2. Instal·lar XAMPP

2.1 Descarregar i instal·lar

# 1. Ves a https://www.apachefriends.org i descarrega XAMPP per Linux (PHP 8.2)
#    O bé, des de la terminal:
cd ~/Descargas
wget "https://sourceforge.net/projects/xampp/files/XAMPP%20Linux/8.2.12/xampp-linux-x64-8.2.12-0-installer.run"

# 2. Donar permisos
chmod +x xampp-linux-x64-8.2.12-0-installer.run

# 3. Executar l'instal·lador (obre una interfície gràfica)
sudo ./xampp-linux-x64-8.2.12-0-installer.run

Segueix l'assistent: accepta les opcions per defecte (instal·la a /opt/lampp).

2.2 Iniciar XAMPP

sudo /opt/lampp/lampp start

Comprova que funciona obrint http://localhost al navegador. Ha d'aparèixer la pàgina de benvinguda de XAMPP.

2.3 Iniciar automàticament en arrancar l'ordinador

sudo nano /etc/systemd/system/xampp.service

[Unit]
Description=XAMPP
After=network.target

[Service]
Type=forking
ExecStart=/opt/lampp/lampp start
ExecStop=/opt/lampp/lampp stop
RemainAfterExit=yes

[Install]
WantedBy=multi-user.target

sudo systemctl daemon-reload
sudo systemctl enable xampp

3. Descarregar el codi

Opció A: Amb Git (recomanada)

cd /opt/lampp/htdocs
sudo git clone https://github.com/oscarbravovillada/BancLLibres.git

Opció B: Còpia de fitxers

sudo mkdir -p /opt/lampp/htdocs/BancLLibres
# Copia el contingut del ZIP a la carpeta anterior
sudo cp -r /ruta/al/zip/descomprimit/* /opt/lampp/htdocs/BancLLibres/

L'estructura de carpetes ha de quedar així:

/opt/lampp/htdocs/BancLLibres/
├── alumnes/            ← fitxa.php, llista.php, nou.php, importar.php
├── classes/            ← classes.php, importar.php
├── exemplars/          ← exemplars.php, nou.php
├── incidencies/        ← index.php
├── llibres/            ← llibres.php, nou.php, importar.php
├── materies/           ← materies.php, importar.php
├── prestecs/           ← index.php, prestecs.php, assignar_lot.php, ...
├── professors/         ← importar.php

│   └── config.php          ← a crear al pas 5

├── docs/
├── src/

├── vendor/

4. Crear la base de dades

# Obrir el client MySQL de XAMPP (password root buit per defecte)
sudo /opt/lampp/bin/mysql -u root

# Dins la consola MySQL:
CREATE DATABASE banc_llibres CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'banc_user'@'localhost' IDENTIFIED BY 'CANVIA_AQUEST_PASSWORD';
GRANT ALL PRIVILEGES ON banc_llibres.* TO 'banc_user'@'localhost';
FLUSH PRIVILEGES;
EXIT;

Ara importa l'esquema:

sudo /opt/lampp/bin/mysql -u banc_user -p banc_llibres \
  < /opt/lampp/htdocs/BancLLibres/database/schema.sql
Quan s'hagi importat, l'usuari administrador per defecte és admin amb contrasenya Admin1234!. Canvia-la al primer accés.

5. Configurar l'aplicació

5.1 Crear config.php

sudo cp /opt/lampp/htdocs/BancLLibres/config/config.example.php \
        /opt/lampp/htdocs/BancLLibres/config/config.php
sudo nano /opt/lampp/htdocs/BancLLibres/config/config.php

6. Permisos de fitxers

# Crear carpeta de PDFs si no existeix
sudo mkdir -p /opt/lampp/htdocs/BancLLibres/uploads/albarans

# Propietari: procés d'Apache (daemon)
sudo chown -R daemon:daemon /opt/lampp/htdocs/BancLLibres

# Permisos generals
sudo chmod -R 755 /opt/lampp/htdocs/BancLLibres

# La carpeta d'uploads necessita escriptura
sudo chmod 777 /opt/lampp/htdocs/BancLLibres/uploads/albarans

7. Verificar la instal·lació

  1. Obre http://localhost/BancLLibres/ al navegador.
  2. Ha d'aparèixer el formulari de login de BancLLibres.
  3. Inicia sessió amb admin / Admin1234!.
  4. Comprova que el panell mostra els menús i les seccions correctament.
  5. Prova de crear un alumne de prova i assignar-li un préstec per comprovar la generació de PDF.

7.1 Comprovació de correus (opcional)

Configura l'adreça de correu d'un alumne de prova, assigna-li un lot i comprova que la família rep el correu.

Si el correu falla, revisa les credencials SMTP a config.php i comprova que el port 587 no estigui bloquejat pel firewall del centre.

8. Primer accés i configuració inicial

  1. Canvia la contrasenya de l'admin: Ves a Usuaris → Editar usuari admin → Canviar contrasenya.
  2. Crea els cursos: Ves a Cursos i afegeix els cursos del centre (1ESO, 2ESO, etc.).
  3. Crea les classes: Per a cada curs, afegeix les classes (1A, 1B, etc.) i assigna el tutor.
  4. Crea les matèries: Ves a Matèries i afegeix les matèries.
  5. Afegeix els llibres: Ves a Llibres i crea el catàleg de llibres.
  6. Crea els exemplars: Per a cada llibre, crea els exemplars físics amb el seu codi.
  7. Crea els lots: Agrupa els exemplars per curs en lots per facilitar l'assignació.
  8. Importa els alumnes: Afegeix els alumnes manualment o via importació CSV (si s'ha implementat).
  9. Crea els professors: Crea els comptes dels professors i assigna'ls a les seves classes.

9. Actualitzar a una nova versió

cd /opt/lampp/htdocs/BancLLibres
sudo git pull origin main

# Si la nova versió inclou canvis d'esquema:
sudo /opt/lampp/bin/mysql -u banc_user -p banc_llibres < database/schema_vX_Y.sql

# Reiniciar Apache si cal:
sudo /opt/lampp/lampp restart
Fes sempre una còpia de seguretat de la BD abans d'actualitzar.

10. Desinstal·lar

# Aturar XAMPP
sudo /opt/lampp/lampp stop

# Eliminar el projecte
sudo rm -rf /opt/lampp/htdocs/BancLLibres

# Eliminar la BD (ATENCIÓ: s'esborra tot!)
sudo /opt/lampp/bin/mysql -u root -e "DROP DATABASE banc_llibres; DROP USER 'banc_user'@'localhost';"

# Desinstal·lar XAMPP (opcional)
sudo /opt/lampp/uninstall
